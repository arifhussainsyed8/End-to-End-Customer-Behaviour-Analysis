🛍️ Customer Shopping Behavior Analysis

An end-to-end data analytics project exploring customer purchasing behavior, product preferences, and customer segments using Python, SQL, and Power BI.

The presentation covers the complete workflow, methodology, dashboard, key insights, and business recommendations.

📊 Project Overview

This analysis is based on 3,900 customer transactions and answers key business questions such as:

* Which customer groups generate the most revenue?
* How do discounts influence purchasing behavior?
* Which products and categories perform best?
* Do subscribers spend more than non-subscribers?

🛠️ Tools Used

* Python (Jupyter Notebook)
* PostgreSQL
* Power BI

🔑 Key Insights

* Young Adults generated the highest revenue.
* Male customers contributed significantly more revenue than female customers.
* Only 27% of customers were subscribers.
* More than 3,100 customers were classified as loyal buyers.
* Several products showed strong dependence on discounts.
